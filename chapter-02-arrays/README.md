# Chapter 2 - Arrays

This chapter covers array concepts in Java and how to store and manipulate multiple values efficiently.

## Topics Covered:
- Introduction to Arrays
- Single Dimensional Arrays
- Multi-Dimensional Arrays
- Array Input and Output
- Traversing Arrays using loops

## Practice Programs:
- Array input and display
- Finding sum of elements
- Maximum and minimum element
- Searching elements in array
- Multi-dimensional array operations

## Purpose:
- Understand how arrays store multiple values
- Learn iteration and data processing
- Build problem-solving skills using arrays

This chapter strengthens the foundation for handling collections of data in Java.
